- **0-hit replacement words to avoid** (checked in KJV.db, deduplicated, whole-word LIKE): smoothly, carpets/carpet, crowd, protestedst (but "protested"/"protest" IS attested — use periphrastic "didst protest"), forsworn, lingerest (but "standest" and "why standest" ARE attested), inflamed (but "swelled" IS attested), astonishest (but "troublest" IS attested), delayest, waitedst, menaced/menacing, strands, stairway, swellest, threats-as-exact-phrase-with-"their" is thin but "threat"(6)/"threatenings"(2) are attested; "threats" itself got 0 hits as a bare LIKE '%threats%' — double check before reusing.
- **Low-but-nonzero (1 hit) words accepted as valid replacements** per project precedent (rule 4 requires nonzero, not high-frequency): usurer (Exodus 22:25), wearisome (Job 7:3), possessest, heldest, exactors (Isaiah 60:17), perjured (1 Tim 1:10, exact phrase "perjured persons").
- **Likely spurious flags** — words retained by nearly every witness with zero divergence, suggesting the auto-flagger caught a normal/common word rather than a genuine rarity or anachronism: tackling (Acts 27:19), tenor (Gen 43:7, Exod 34:27), tentmakers (Acts 18:3 — also CUR "craftsmen" duplicates "craft" already in verse), thicker (1 Kings 12:10, II Chron 10:10), threefold (Eccl 4:12), thirdly (1 Cor 12:28 — ordinal in a first/secondarily/thirdly list, replacing it breaks the enumeration), theft (Exodus 22:3-4 — also "robbery" wrongly implies violence not in the text). Recommend NO-SAFE-SWAP with this note whenever nearly all witnesses agree on the flagged word.
- **Proper nouns wrongly flagged as rare words**: "taverns" in "The Three Taverns" (Acts 28:15, Tres Tabernae) is part of a place name, not a common noun — NO-SAFE-SWAP.
- **Ambiguous/disputed-identity words**: "swan" (Lev 11:18, Deut 14:16) — unclean-bird list identifications are disputed among all witnesses (Darby keeps swan, ASV has horned owl, BSB has osprey); DRC's Deut 14:16 lists both "swan" AND "stork" as separate items, so swapping swan→stork would conflate two list entries. NO-SAFE-SWAP.
- **Duplicate-word traps found**: CUR "tapestry→herself" duplicated the verse's own subject "herself" (Prov 7:16, 31:22) — fixed with "hangings" (attested, Exodus 26 tabernacle term). CUR "talkers→tongues" pattern and "throng→narrow" pattern both needed checking that the replacement word wasn't already present later in the same verse (e.g. Luke 8:45 "throng...press" — "press" already used later, so used "crush" instead, matching ASV).
- **Meaning-inversion traps found**: "tenderhearted→fainthearted" is CORRECT for II Chron 13:7 (young/timid Rehoboam — Darby's own witness there is "faint-hearted") but WRONG for Eph 4:32 (compassion, not cowardice — fixed to "pitiful," matching 1 Peter 3:8's "be pitiful, be courteous" construction). Same source word, opposite correct rulings depending on verse sense — always check context, not just the lemma.
- **Disguised re-inflection pattern**: several CUR choices merely stripped a suffix from the flagged word rather than genuinely replacing it — "thereabout→about", "thereinto→therein", "thicket→thick", "terrifiest→terrify", "testifiedst→testified" all do this and were ruled invalid per rule 3, requiring a genuinely different lexeme instead (amazed, into it, bush/den, troublest, didst protest).
