# Review: current uses of “hail” (greeting vs. weather)

*68 verses currently contain a **hail** form. Grouped below. Only the greeting uses are candidates for **greet**; the weather/plague ‘hail’ and the famous direct-address ‘Hail’ are left as-is per the rule you gave (‘All hail, King of the Jews’ stays; interpersonal greeting → greet/greeteth, which implies welcoming in).*

- **34** interpersonal greetings (base was *salute/congratulate*, changed toward *hail*) → **suggest greet/greeteth**
- **5** famous direct-address *Hail* acclamations → **keep hail**
- **29** weather/plague *hail* (or the name Ben-hail) → **not greetings, leave**
